FHIR® tutorials

This set of pages contains a series of FHIR tutorials for those just beginning to learn the new specification. The tutorials require no prior knowledge of FHIR or REST.

Please note these tutorials have been written against a version of FHIR, based on the FHIR R4.

To continue, use the tabs left of the page and work through the tutorials from top to bottom starting with the Simple Patient.
